Your goal:

1. Read the provided raw text carefully.

2. Identify all SKILLS: technical abilities, soft skills, tools, or domain-specific knowledge.

3. Identify all COMPETENCIES: broader areas of professional ability or expertise.

1. Extract ONLY competencies that are EXPLICITLY stated in human-readable text.

   - JSON keys, field names, IDs, timestamps, URLs, tokens, emails

   - Any inferred system capability (e.g. "state management", "coordination")

   - Programming languages (e.g. JavaScript, Python)
   - Frameworks & libraries (e.g. React, Node.js)
   - Tools & platforms (e.g. Docker, GitHub)
   - Clearly named professional domains (e.g. Backend Development)
   - Clearly stated soft skills (e.g. Team Leadership)

5. Return ONLY unique competencies.

6. If NO explicit competencies are found, return an EMPTY array [].

Output format:
- Respond ONLY with a valid JSON array of strings.
- No explanations, no comments, no markdown.

{{raw_text}}
